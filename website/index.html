开源免费 · Tauri 2 + React · v1.6.0

极致打磨的智能剪贴板管理工具。自动记录文本、图片和文件历史，内置 AI 摘要翻译与多服务问答面板，搭配代码高亮、富文本编辑与 Markdown 渲染，端到端加密云同步。

为高阶效率而生

捕获、整理、编辑、分享剪贴板内容 — 一切由 AI 驱动。

智能剪贴板

自动捕获文本、图片、富文本和文件。SHA-256 去重、即时搜索、类型筛选、智能内容分类、19 种编程语言检测。

AI 集成

多服务商 AI 支持（OpenAI、Claude、Gemini、DeepSeek、Ollama）。一键「问 AI」将内容发送到任意 AI 服务，支持自定义预设。自动摘要和 AI 翻译。

富文本编辑

CodeMirror 6 代码编辑器，支持 12+ 种语言语法高亮、字体大小调节和工具栏。TipTap 2 所见即所得编辑器。可拖拽分屏视图，编辑与预览同步。

Markdown 渲染

支持 GFM 语法、KaTeX 数学公式、Shiki 代码块高亮和 Mermaid 流程图。智能自动检测内容类型（代码/Markdown/富文本/纯文本）。

更多特性

加密云同步

AES-256-GCM 端到端加密。支持 WebDAV、Google Drive、OneDrive 同步。跨设备设置同步。

多引擎翻译

百度、有道、Google 翻译引擎，以及 AI 翻译模式。语言自动检测，一键复制。

Ask AI 面板

浏览器风格多标签页 AI 面板。ChatGPT、Claude、Gemini、DeepSeek、Kimi、豆包等。每个服务独立会话。

标签管理

完整标签系统，支持创建、重命名、删除。置顶重要条目、收藏、按标签或类型筛选。

来源追踪

显示每条记录的来源应用名称、图标和 URL。点击可打开原始来源链接。

模板与 OCR

支持变量替换的可复用文本模板。内置 OCR 从图片中提取文字（Windows）。

智能窗口

所有窗口记忆尺寸。智能定位到光标附近。全局快捷键、方向键导航、快速粘贴。

个性化

深色/浅色/跟随系统主题。自定义字体和界面缩放。中英双语。开机自启。

隐私优先

所有数据本地存储。无遥测、无追踪。可选加密同步 — 数据始终归你所有。

技术栈

Framework Tauri 2
Frontend React 19 + TypeScript
Backend Rust
样式 Tailwind CSS 4
代码编辑器 CodeMirror 6
富文本编辑器 TipTap 2
Markdown Shiki + KaTeX + Mermaid
