## 1. 项目概述

### 1.1 产品定位

AlgerClipboard 是一款面向个人用户的跨平台智能剪贴板管理工具。通过全局快捷键快速调出，支持剪贴板历史管理、多格式内容处理、图片标注、文本翻译、常用文本模板以及跨设备云同步，旨在提升用户日常复制粘贴的效率和体验。

### 1.2 目标平台

| 平台    | 最低版本要求        |
| ------- | ------------------- |
| Windows | Windows 10 (1803+)  |
| macOS   | macOS 11 (Big Sur+) |
| Linux   | Ubuntu 20.04+ / 主流发行版（X11 & Wayland） |

### 1.3 目标用户

通用个人用户，包括但不限于：
- 日常办公用户：频繁复制粘贴文本、图片
- 开发者：代码片段管理、常用命令模板
- 设计师：截图标注、图片管理
- 多设备用户：需要在不同电脑间同步剪贴板

## 2. 功能需求

### 2.1 剪贴板监听与历史管理

#### 2.1.1 剪贴板监听
- **FR-001**：实时监听系统剪贴板变化，自动记录每次复制操作
- **FR-002**：支持的内容类型：纯文本、富文本 (RTF/HTML)、图片、文件路径
- **FR-003**：自动记录来源应用名称
- **FR-004**：通过内容 hash 自动去重，重复内容只更新时间戳

#### 2.1.2 历史管理
- **FR-005**：历史记录上限可配置，默认 1000 条
- **FR-006**：超过上限时自动清理最旧的非收藏条目
- **FR-007**：支持手动删除单条或批量删除
- **FR-008**：支持按关键词全文搜索
- **FR-009**：支持按内容类型（文本/图片/文件）筛选
- **FR-010**：支持按时间范围筛选
- **FR-011**：支持按标签筛选

### 2.2 快捷键与窗口管理

#### 2.2.1 全局快捷键
- **FR-012**：全局快捷键呼出/隐藏主面板（默认 `Ctrl+Shift+V` / `Cmd+Shift+V`）
- **FR-013**：所有快捷键用户可自定义
- **FR-014**：支持快捷键直接粘贴最近 N 条记录（默认 `Ctrl+1~9`）
- **FR-015**：支持快捷键触发翻译当前条目

#### 2.2.2 窗口管理
- **FR-016**：面板以悬浮窗口形式展示，可跟随鼠标位置弹出
- **FR-017**：支持固定窗口位置（钉在屏幕上）
- **FR-018**：点击面板外区域或按 `Esc` 自动关闭面板
- **FR-019**：支持窗口透明度调节

### 2.3 智能填充

- **FR-020**：选中条目后按 Enter，自动将内容填充到当前活动窗口的输入焦点
- **FR-021**：文本内容支持纯文本粘贴和富文本粘贴两种模式
- **FR-022**：图片条目填充为图片粘贴（支持的应用）或文件路径
- **FR-023**：文件条目填充为文件路径文本
- **FR-024**：填充后面板自动关闭（可配置为不关闭）

### 2.4 图片标注编辑器

- **FR-025**：双击图片条目进入标注编辑器
- **FR-026**：支持裁剪工具
- **FR-027**：支持画笔涂鸦（可调颜色、粗细）
- **FR-028**：支持箭头绘制
- **FR-029**：支持矩形/椭圆框选标注
- **FR-030**：支持文字标注（可调字号、颜色）
- **FR-031**：支持马赛克/模糊遮挡
- **FR-032**：支持撤销/重做操作
- **FR-033**：编辑完成后可保存为新条目或覆盖原条目
- **FR-034**：支持导出为 PNG/JPG 格式

### 2.5 翻译模块

#### 2.5.1 翻译引擎
- **FR-035**：支持百度翻译 API
- **FR-036**：支持有道翻译 API
- **FR-037**：支持谷歌翻译 API
- **FR-038**：用户可在设置中配置各引擎的 API Key / Secret
- **FR-039**：支持设置默认翻译引擎
- **FR-040**：某引擎失败时可自动降级到备选引擎

#### 2.5.2 翻译触发
- **FR-041**：手动触发：选中文本条目后点击翻译按钮或快捷键
- **FR-042**：自动触发：检测到剪贴板中的外语文本时，自动弹出翻译结果（可开关）
- **FR-043**：支持设置源语言和目标语言
- **FR-044**：翻译结果可一键复制或保存为新条目

### 2.6 收藏与模板

#### 2.6.1 收藏
- **FR-045**：任意条目可标记/取消标记为收藏
- **FR-046**：收藏条目不受历史上限自动清理影响
- **FR-047**：收藏条目独立的筛选视图

#### 2.6.2 模板
- **FR-048**：支持创建常用文本模板
- **FR-049**：模板支持变量占位符：`{date}`, `{time}`, `{datetime}`, `{clipboard}`（当前剪贴板内容）
- **FR-050**：模板支持分组/分类管理
- **FR-051**：选择模板时自动替换变量并填充

### 2.7 跨设备同步

#### 2.7.1 同步方式
- **FR-052**：支持 WebDAV 同步（兼容坚果云、NextCloud、自建 WebDAV 等）
- **FR-053**：支持 Google Drive 同步
- **FR-054**：支持 OneDrive 同步
- **FR-055**：云存储适配器采用插件式架构，可后续扩展更多存储方式

#### 2.7.2 同步策略
- **FR-056**：增量同步，仅传输变更数据
- **FR-057**：同步频率可配置（实时 / 定时 / 手动）
- **FR-058**：冲突策略：时间戳优先，保留冲突副本供用户手动处理
- **FR-059**：离线可用，联网后自动同步积压数据
- **FR-060**：同步状态实时展示（同步中/已同步/同步失败）
- **FR-061**：大文件（图片/文件）分块压缩上传

#### 2.7.3 数据安全
- **FR-062**：同步数据支持端到端加密（可选，用户设置加密密钥）
- **FR-063**：API Key 等敏感信息本地加密存储，不同步

### 2.8 插件系统

- **FR-064**：支持安装/卸载/启用/禁用第三方插件
- **FR-065**：插件可订阅生命周期事件：`onClipboardChange`, `onBeforePaste`, `onSync` 等
- **FR-066**：插件可注册自定义 UI 面板
- **FR-067**：插件可添加右键上下文菜单项
- **FR-068**：前端插件通过 iframe 沙箱隔离运行
- **FR-069**：插件包含 `manifest.json` 描述元信息（名称、版本、权限声明）
- **FR-070**：云存储适配器作为内置插件发布

### 2.9 设置与个性化

- **FR-071**：支持开机自启动
- **FR-072**：支持多语言（至少中文、英文）
- **FR-073**：支持主题切换：亮色/暗色/跟随系统
- **FR-074**：数据存储目录可自定义
- **FR-075**：支持数据导出/导入（备份恢复）

## 3. 非功能需求

### 3.1 性能

- **NFR-001**：剪贴板变化检测延迟 < 200ms
- **NFR-002**：面板呼出响应时间 < 300ms
- **NFR-003**：搜索 1000 条记录响应时间 < 100ms
- **NFR-004**：内存占用空闲时 < 50MB
- **NFR-005**：安装包体积 < 20MB

### 3.2 安全

- **NFR-006**：API Key 使用系统级密钥管理存储（macOS Keychain, Windows Credential Manager, Linux Secret Service）
- **NFR-007**：插件运行在沙箱中，不可直接访问文件系统
- **NFR-008**：同步传输使用 HTTPS/TLS 加密

### 3.3 可靠性

- **NFR-009**：本地数据库损坏时自动修复或降级运行
- **NFR-010**：同步失败不影响本地功能正常使用
- **NFR-011**：应用崩溃后自动恢复上次状态

### 3.4 可用性

- **NFR-012**：首次启动提供引导向导
- **NFR-013**：支持系统托盘图标常驻
- **NFR-014**：关键操作提供 Toast 通知反馈

## 4. 用户交互流程

### 4.1 核心使用流程

```
复制内容 → 自动记录到历史 → 按快捷键呼出面板 → 搜索/浏览条目 → 选中条目 →
Enter 填充到当前应用 → 面板自动关闭
```

### 4.2 翻译流程

```
复制外语文本 → (自动模式) 弹出翻译结果 / (手动模式) 呼出面板 → 选中条目 →
点击翻译 → 选择引擎和目标语言 → 查看结果 → 复制或保存
```

### 4.3 图片标注流程

```
复制/截图图片 → 呼出面板 → 双击图片条目 → 进入标注编辑器 →
添加标注 → 保存 → 填充或分享
```

### 4.4 同步流程

```
设置中配置云存储账户 → 开启同步 → 本地变更自动上传 →
其他设备拉取变更 → 冲突时保留双方副本 → 用户手动处理冲突
```

## 5. 优先级规划

### P0 - 核心功能（MVP）
- 剪贴板监听与历史管理 (FR-001 ~ FR-011)
- 全局快捷键与窗口管理 (FR-012 ~ FR-019)
- 智能填充 (FR-020 ~ FR-024)
- 收藏功能 (FR-045 ~ FR-047)
- 基础设置 (FR-071 ~ FR-075)

### P1 - 重要功能
- WebDAV 同步 (FR-052, FR-056 ~ FR-063)
- 翻译模块 (FR-035 ~ FR-044)
- 模板功能 (FR-048 ~ FR-051)
- 图片标注编辑器 (FR-025 ~ FR-034)

### P2 - 增强功能
- Google Drive / OneDrive 同步 (FR-053 ~ FR-055)
- 插件系统 (FR-064 ~ FR-070)
- 翻译自动降级 (FR-040)
- 翻译自动检测 (FR-042)

## 6. 约束与假设

### 6.1 约束
- 不使用后端服务器，所有同步通过第三方云存储完成
- 翻译 API 需要用户自行申请 Key
- 插件系统 V1 仅支持前端插件（JavaScript），WASM 插件为远期目标

### 6.2 假设
- 用户有可用的 WebDAV / Google Drive / OneDrive 账户（同步功能可选）
- 用户网络环境可正常访问所配置的翻译 API
- 各平台系统 API 对剪贴板监听的能力基本一致
